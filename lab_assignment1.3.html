Function of the <img> Tag in HTML

Theory

The <img> tag in HTML is used to display images on a web page. Images help make web pages more attractive and improve user understanding of the content.

Common Attributes of <img> Tag

1. src Attribute

The src attribute specifies the source or path of the image file. Without the src attribute, the image cannot be displayed on the web page.

2. alt Attribute

The alt attribute provides alternative text for the image. This text is shown if the image cannot be loaded and is also helpful for screen readers used by visually impaired users.

Conclusion

The <img> tag is essential for adding images to web pages. Attributes like src and alt are important for displaying images correctly and improving accessibility.

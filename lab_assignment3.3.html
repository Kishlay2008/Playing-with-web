Document Object Model (DOM)

Theory

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree structure. In the DOM, each HTML element is treated as an object that can be accessed and manipulated using JavaScript. The DOM allows us to change the content, structure, and style of a web page dynamically.

Selecting Elements Using DOM

JavaScript can select HTML elements using methods such as getElementById, getElementsByClassName, and querySelector. Once selected, the elements can be modified.

innerText

The innerText property is used to change or retrieve only the text content of an HTML element. It does not include HTML tags.

innerHTML

The innerHTML property is used to change or retrieve the HTML content inside an element. It can include HTML tags.

Event Delegation and addEventListener

Theory

1. Event Delegation:

Event delegation is a technique where a parent element listens for events from its child elements. Instead of adding separate event listeners to each child, the parent detects which child triggered the event. This reduces memory usage and works for dynamically added elements.

2. addEventListener:

The addEventListener method is used to attach an event handler to an HTML element. It allows specifying the event type, the function to execute, and optional settings like capturing or once. It is more flexible than older methods like onclick.

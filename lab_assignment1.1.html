Use of Attributes in HTML

Theory

HTML attributes are used to provide additional information about HTML elements. Attributes are written in the opening tag and usually follow the format name="value". They help control the behavior and appearance of elements.

Commonly Used HTML Attributes

1. id Attribute

The id attribute is used to uniquely identify an HTML element.

2. href Attribute

The href attribute is used with the anchor tag to specify the URL of the linked page.

3. src and alt Attributes

The src attribute specifies the source of an image. The alt attribute provides alternative text if the image cannot be displayed.

Conclusion

HTML attributes enhance the functionality of web pages by adding extra information to HTML elements. Attributes like id, href, src, and alt are commonly used in HTML.
